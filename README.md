# Биржевой стакан

Проект представляет собой многопоточный сервис обработки торговых данных с реализацией упрощенного Order Book. Сервис принимает поток событий, обрабатывает их и рассылает клиентам обновления лучшего уровня.

## Микросервис Order Book

- принимает поток заявок
- формирует книгу заявок (bid/ask book)
- обрабатывает ордера (New, Cancel, Replace)
- рассчитывает лучшую цену (best bid / best ask)
- предоставляет подписку на обновления стакана по TCP
- публикует обновления в реальном времени

## Клиентское приложение

- поддерживает консольный интерфейс
- отображает текущий топ уровня стакана (best bid/ask)
- получает обновления по подписке
