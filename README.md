класс GraphicalUI будет принимать указатель на объект client lib в (дальнейшем Объект) в конструкторе, запускать QAplication через статический метод run(int argc, char* argv[]), вызывать необходимые методы Объекта в зависимости от активировнных элементов управления, передавая ему необходимую информацию (GrapghicalUI должен гарантировать корректность информации поступающей в качетсве аргументв в методы Объекта). Проверять корректность в части способности к отображению информацию возвращаемую методами Объекта и отображать ее в удобоваримом для пользователя виде.
