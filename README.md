Реализован TCP client с подключением к серверу по заданному IP/port.
Добавлена отправка и прием сообщений по TCP-соединению, вывод ответов в консоль.
Обработаны базовые ошибки сети и сценарии разрыва соединения.

Стек технологий
- Язык: C++20
- Сборка: CMake 3.15
- Boost: filesystem, system, serialization
- Тесты: Catch2
